# 画像に旗立てBeach_flag

<h3>画像分類支援ソフトの</h3>

このソフトは物体検知のDeep Learningの未検地や誤検知などに分類する作業を改善するために開発しました。<br>
動画を画像に変換した大量の画像を同時に分類することができます。<br><br>
分類は同時に10種類まで分けることができます。<br><br>
ソースはあんまりきれいに書いてないので見づらいと思います。
詳しくはQiitaに記事を書きます。<br>
