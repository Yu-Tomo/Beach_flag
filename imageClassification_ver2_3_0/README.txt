分類を始めるには、上方の　「開く」から分類したい画像のディレクトリを選択します。

「開いてるファイル」には現在表示しているファイル名を表示します。

画像が表示されるので、上下スクロールで画像を移動できます。

分類にはモードが二つあります。

デフォルトで個別分類モードになっています。

個別分類モードでは、一つ一つ画像を見て分類する必要があります。

連続分類モードでは、現在のラジオボタンで選択した分類が、スクロール時、次に表示される画像にも適応されます。

よって、分類を変化させずに、移動したいときは個別分類モードを使用します。


「分類を結果を保存する」では、新しくwindowが表示され、誤検知の画像を入れるディレクトリと未検知の画像を入れるディレクトリを選択できます。

「保存する」を押すと保存できます。

「閉じるを押す」と画像を閉じることができます。


黄色の枠は、今後何かメッセージを出せるように用意しています。
